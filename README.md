# Segurança-de-Redes-e-da-Informação
Fundamentos da segurança da informação, começando pelos conceitos básicos sobre proteção de dados.

O texto explora os fundamentos da Segurança da Informação, destacando como o crescimento exponencial de dispositivos e da internet torna a segurança dos dados transmitidos um grande desafio. A essência do conceito é definida como um conjunto de práticas e recursos voltados para proteger ativos e dados sigilosos contra riscos internos e externos. O cerne desta proteção é o Triângulo da Segurança da Informação, que é composto pelos três pilares essenciais: Confidencialidade, garantindo acesso apenas a usuários autorizados; Integridade, assegurando que os dados não sejam alterados de forma fraudulenta; e Disponibilidade, mantendo os dados acessíveis sempre que necessário. Finalmente, o texto alerta para diversos riscos operacionais e ameaças, como ataques de negação de serviço e ransomware, enfatizando a importância de medidas de segurança adequadas e atualizações constantes para a continuidade dos negócios.
